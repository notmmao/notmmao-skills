# yt-audio-download 音频下载技能

## 描述
基于 yt-dlp 实现的通用视频平台音频下载技能，自动将下载的音频转换为 192kbps 高质量 MP3 格式，内置 URL 去重逻辑避免重复下载，支持明确URL下载和模糊歌名搜索下载。

## 触发条件
当用户发送以下类型请求时自动触发：
- 「下载这个url对应的音频」
- 「下载[URL]的音乐」
- 「提取这个视频的音频」
- 任何包含「下载音频/音乐」+ 视频URL的请求
- 模糊请求：「下载[歌手名]的[歌名]」、「下载[歌手名]的新歌」等无明确URL的音频下载请求

## 处理流程
1. **有明确URL场景**：直接调用下载脚本下载对应音频
2. **无明确URL场景**：
   - 第一步：搜索确认对应歌曲的正式名称，避免下载非官方/非歌曲内容
   - 第二步：搜索B站上对应歌曲的官方MV/完整纯享音频视频链接，优先选择官方发布、时长匹配的完整歌曲内容，过滤介绍类、资讯类视频
   - 第三步：调用下载脚本下载选中的视频音频

## 使用方式
```bash
python3 scripts/yt_dlp_demo.py <目标视频URL>
```

## 特性
✅ **自动去重**：以URL为唯一标识，相同URL仅会下载一次，已下载内容会直接返回本地路径
✅ **统一存储**：所有下载文件保存在 `<cache>/yt-audio-download` 目录
✅ **下载记录**：历史下载记录保存在 `<cache>/yt-audio-download/.downloaded.json`
✅ **易迁移**：脚本与技能配置文件同目录，可直接打包整个技能目录进行迁移
✅ **自动转码**：默认输出192kbps MP3格式，适配绝大多数播放场景
✅ **防错逻辑**：模糊搜索场景自动过滤介绍类、资讯类非歌曲视频，确保下载内容是完整歌曲音频
