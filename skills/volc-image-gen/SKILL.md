# 火山引擎图片生成 Skill
基于火山引擎豆包Seedream系列模型的AI绘画技能，支持文生图、单参考图生图、多参考图生图三种模式，生成高清电影级画质图片。

## 触发场景
当用户说以下内容时触发此 Skill：
- 生成图片 [描述]
- 文生图 [描述]
- 图生图 [描述] 参考图=[URL]
- 画图 [描述]
- AI绘画 [描述]
- 帮我画一张 [描述]

## 功能特性
✅ 支持文生图：纯文字描述生成高清图片
✅ 支持图生图：传入参考图+文字描述生成新图片
✅ 支持多参考图生图：最多传入3张参考图，实现风格迁移、元素融合等效果
✅ 支持2K/4K超清分辨率
✅ 可选水印开关
✅ 支持自定义模型版本（默认使用最新Seedream 4.0模型）
✅ 默认自动缓存生成的图片到 ~/.cache/volc-image-gen/ 目录，自动去重命名

## 前置配置
请确保在 `/home/ota/.openclaw/.env` 文件中配置相关参数：
```env
# 必填：火山引擎API密钥
ARK_API_KEY=你的火山引擎API密钥
# 可选：默认使用的模型版本，不配置则默认使用 doubao-seedream-4-0-250828
ARK_API_MODEL=doubao-seedream-5-0-260128
```

## 使用方式
### 1. 基础文生图
```
生成图片 [提示词]
```
示例：`生成图片 星际穿越，黑洞，黑洞里冲出一辆快支离破碎的复古列车，电影大片质感`

### 2. 自定义参数
```
生成图片 [提示词] 尺寸=4K 无水印 模型=doubao-seedream-5-0-260128
```
可选参数：
- 尺寸：2K（默认）/4K/1024x1792等
- 无水印：添加此参数关闭默认水印
- 模型：指定模型版本，默认`doubao-seedream-4-0-250828`

### 3. 图生图（单参考图）
```
图生图 [提示词] 参考图=[图片URL]
```
示例：`图生图 生成狗狗趴在草地上的近景画面 参考图=https://example.com/dog.png`

### 4. 多参考图生图
```
图生图 [提示词] 参考图=[URL1] 参考图=[URL2]
```
示例：`图生图 将图1的服装换为图2的服装 参考图=https://example.com/1.png 参考图=https://example.com/2.png`

### 5. 保存图片到本地
```
生成图片 [提示词] 保存=./output.png
```

## 实现逻辑
1. 解析用户输入的提示词和参数
2. 加载`.env`中的`ARK_API_KEY`环境变量
3. 调用`volc_image_gen.py`脚本执行生成请求
4. 返回生成的图片URL、尺寸，若指定保存路径则自动下载到本地
5. 可直接通过`MEDIA:`指令将生成的图片发送给用户
