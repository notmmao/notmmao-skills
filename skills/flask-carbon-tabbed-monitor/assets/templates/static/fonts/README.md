# IBM Plex 字体文件

将以下 woff2 字体文件放在此目录中：

| 文件名 | 字体 | 权重 |
|--------|------|------|
| `ibm-plex-mono-400.woff2` | IBM Plex Mono | Regular (400) |
| `ibm-plex-mono-600.woff2` | IBM Plex Mono | SemiBold (600) |
| `ibm-plex-sans-latin.woff2` | IBM Plex Sans | Variable (100-900) |

## 下载来源

- IBM Plex Mono: https://github.com/IBM/plex/tree/master/packages/plex-mono
- IBM Plex Sans: https://github.com/IBM/plex/tree/master/packages/plex-sans

从 GitHub Releases 下载对应 woff2 文件即可。Latin 子集文件足够覆盖英文 + 基本符号。
